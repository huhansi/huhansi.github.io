58. 最后一个单词的长度

Posted on 2018-09-04 | In leetcode

题目

给定一个仅包含大小写字母和空格 ' ' 的字符串，返回其最后一个单词的长度。

如果不存在最后一个单词，请返回 0 。

说明：

一个单词是指由字母组成，但不包含任何空格的字符串。

示例1：

解法

解法一：

题目的意思就是找到最后一个非空格序列的长度，那么首先去掉首尾的空格，然后找到最后一个空格出现的索引。如果索引为-1，说明剩下的序列中没有空格，直接返回该序列的长度即可，否则，返回序列长度减去1减去索引的值就行。
